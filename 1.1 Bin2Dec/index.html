1.1 Bin2Dec

This is project 1.1 of the app-ideas challenge, details are available on the Bin2Dec challenge page

User Stories

  • User can enter up to 8 binary digits in one input field
  • User must be notified if anything other than a 0 or 1 was entered
  • User views the results in a single output field containing the decimal (base 10) equivalent of the binary number that was entered

Bonus features

  • User can enter a variable number of binary digits

You can read all about the app-ideas challenge and see a list of the individual challenges on the main app-ideas repo
