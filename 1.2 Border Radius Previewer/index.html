1.2 Border Radius Previewer

This is project 1.2 of the app-ideas challenge, details are available on the Border Radius Previewer challenge page

My attempt at this project is available in my main Github repo for the whole challenge, but specifically in the folder for this particular challenge

User Stories

  • User can see a box which has a border-radius property applied to it
  • User can change the 4 border-radius values that are applied to the box (top-left, top-right, bottom-left, bottom-right)
  • User can copy the resulting CSS to the clipboard

Bonus features

  • User can change all 8 possible values of the border-radius in order to create a complex shape

You can read all about the app-ideas challenge and see a list of the individual challenges on the main app-ideas repo
